## 🎨 Tema Oscuro de Galaxia

### Colores Principales
- **Negro profundo:** `#0a0e27` - Fondo principal
- **Púrpura místico:** `#a855f7` - Accents principales
- **Rosa neón:** `#ec4899` - Highlights
- **Morado oscuro:** `#9333ea` - Textos secundarios
- **Blanco helado:** `#e9d5ff` - Texto principal

### Efectos Especiales
✨ Fondo animado con galaxia
⭐ Estrellas parpadeantes
🌀 Gradientes dinámicos
💫 Efectos hover suaves
🔥 Sombras con brillo neón

---

## 📊 Características Visuales

### Estado de Conexión
```
🟢 En línea - Usuario conectado
✗ Desconectado - Sesión cerrada
⚡ Sistema - Notificaciones
```

### Tipos de Mensajes
```
✦ Fantasma: Mensaje normal (de otro usuario)
✦ Fantasma: Mensaje mío (derecha, resaltado)
⚡ SISTEMA: Notificación (centrado, tenue)
```

### Acciones
```
CONECTAR - Botón de login
ENVIAR   - Enviar mensaje
✕ CERRAR - Cerrar sesión y borrar todo
```

## ⚙️ Configuración Visual

Para cambiar colores, editar `client/src/App.css`:

```css

border-color: #6b21a8;    /* Bordes */
background: #a855f7;     /* Fondos */
color: #e9d5ff;          /* Texto principal */
color: #ec4899;          /* Texto secundario */
```
